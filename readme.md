# Wallet App 

## Introdução 


Este é um painel de gerenciamento financeiro feito com JavaScript, HTML e CSS. O objetivo principal é
que o usuário adicione e exclua lançamentos financeiros (que são salvos na API) e informe o saldo
do mês.
